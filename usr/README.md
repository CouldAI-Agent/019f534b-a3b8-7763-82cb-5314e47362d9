# Dhaloota Nageenyaa

Appilikeeshiniin kun yaada fi aadaa nageenyaa, tokkummaa, fi jaalala walii keenyaa cimsuuf kan qophaa'edha. Yaada ijoo: “Dhaloonni kun mal taʼet wal ciraa osoo taʼee yoona in tokkoomu moo Dhagaa owwalcha walitti guuru” jedhu irratti hundaa'uun yaadannoo garaagaraa of keessatti qabata.

## Amaloota Ijoo
*   **Yaadannoo Nageenyaa:** Ergaawwan tokkummaa fi nageenyaa dubbisuuf.
*   **Saxaxa Miidhagaa:** Bilbila, taableetii fi kompiitara irraatti sirriitti kan mul'atu.
*   **Fayyadamuuf Salphaa:** Salphaatti dubbisuuf kan mijate.

## Teeknooloojii Fayyadamne
*   Flutter
*   Dart
*   Material Design 3

## Akkamitti Hojjechiisuun Danda'ama
1. Appilikeeshinicha buufadhu ykn karaa GitHub clone godhi.
2. `flutter pub get` galchuun packages barbaachisoo ta'an guuti.
3. `flutter run` jedhiitii hojjechiisi.

## Waa'ee CouldAI
Appilikeeshiniin kun [CouldAI](https://could.ai) fayyadamuun kan ijaarame dha. CouldAI'n ijaaraa appilikeeshinii AI dhaan deeggarame yoo ta'u, yaadawwan adda addaa gara appilikeeshinii dhugaa iOS, Android, Web fi Desktop tti jijjiira. Kanas kan godhu AI ofiin of bulchu fayyadamuun yoo ta'u, isaaniis appilikeeshinoota haala gaariin ijaaruu, qoruu, tamsaasuufi fooyyessuu danda'u.
